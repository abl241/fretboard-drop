3. MAKE FRETBOARD HEIGHT RESPONSIVE AND INTERNALLY CONSISTENT

Replace the fragile combination of:

- fixed outer height
- hard-coded internal height subtraction
- repeated minimum row heights
- hidden overflow

with a simple responsive layout whose dimensions agree with each other.

Preferred direction:

- give the neck a bounded responsive height or aspect ratio

- use clamp(), aspect-ratio, grid, or flex only when they simplify the geometry

Do not create a complex JavaScript resize engine unless CSS cannot solve the issue reliably.

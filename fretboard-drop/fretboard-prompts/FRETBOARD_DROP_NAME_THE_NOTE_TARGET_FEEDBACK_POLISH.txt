PRIMARY GOAL

Improve the active Name the Note gameplay loop so it feels more polished, exciting, game-like, and App Store-ready without changing the core rules.

The current layout is cleaner and responsive, but the moment-to-moment feedback still feels like a quiz form rather than a premium learning game.

The weakest element is the repeated draining cyan fill inside all seven A-G answer buttons. It looks rudimentary, duplicates the same countdown seven times, and pulls attention away from the actual fretboard target.

The better interaction model is:

target appears → pressure builds around the target → player answers → clear feedback → next target

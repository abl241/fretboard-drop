Name the Note evidence should include:

Recommended evidence mapping:

- `correct` = successfully completed targets
- `wrong` = total incorrect attempts
- `misses` = timeouts
- `accuracy` = `correct / (correct + wrong attempts + timeouts) * 100`
- `bestStreak` = best first-try streak
- `responseProgress` = `responseMs / totalQuestionMs`
